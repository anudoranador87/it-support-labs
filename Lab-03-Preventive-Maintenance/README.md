# Lab 03: Preventive Maintenance — Thermal & Keyboard Issues

**CompTIA A+ Domain:** 220-1201, Domain 1.5 (Maintenance & Safety)  
**Level:** Intermediate  
**Time:** 40 minutes  
**Real hardware:** Laptop with overheating + sticky keyboard

---

## 🎯 Objective

- Perform thermal maintenance (check temps, clean vents)
- Address keyboard issues (stuck keys, cleaning)
- Document preventive maintenance procedures
- Follow safety protocols (ESD, power down)

---

## 📋 Scenario

**Ticket:** "Laptop getting hot, fan running loud. Spacebar sometimes doesn't respond."

Your job: Clean vents, check thermals, fix keyboard.

---

## 🔧 Tools & Methods

| Tool | Purpose |
|------|---------|
| Compressed air | Clean vents & keyboard |
| Thermal imaging | Check heat distribution |
| System monitoring | Read CPU/GPU temps |
| Anti-static wrist strap | Prevent ESD |
| Documentation | Before/after photos |

---

## 📸 Step-by-Step Walkthrough

### **Step 1: Safety First**

- Power down completely (no sleep mode)
- Wait 5 minutes for cooling
- Put on anti-static wrist strap
- Disconnect power adapter

### **Step 2: External Cleaning**

- Use compressed air on vents (short bursts)
- Blow debris away from device
- Clean keyboard with soft brush

**Before:** Vents clogged with dust  
**After:** Clean, clear vents

### **Step 3: Check Thermals**

- Boot into BIOS/monitoring software
- Run 5-minute stress test
- Record CPU/GPU temperatures
- Expected: 60-75°C under load

**Before:** 95°C (overheating)  
**After:** 72°C (normal)

### **Step 4: Test Keyboard**

- Type each key multiple times
- Spacebar specifically (was sticking)
- Result: All keys responsive
