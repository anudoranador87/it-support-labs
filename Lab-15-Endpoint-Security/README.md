# Lab 15: Endpoint Security — Malware & Defender

**CompTIA A+ Domain:** 220-1201, Domain 3.1  
**Level:** Intermediate  
**Time:** 40 minutes  
**Scenario:** A front desk computer has been acting slow, and a suspicious popup appeared. You need to investigate and secure the endpoint.

---

## 🎯 Objective

- Understand malware types and symptoms
- Run Windows Defender scans and review quarantined files
- Isolate a potentially infected machine from the network
- Verify browser extensions and startup programs

---

## 📸 Step-by-Step Walkthrough

### **Step 1: Isolate the Machine**
1. [Disable the network adapter so the infection doesn't spread]

### **Step 2: Check Startup and Extensions**
1. [Open Task Manager > Startup and look for suspicious executables]
2. [Check Chrome/Edge extensions for adware]

### **Step 3: Run Windows Defender**
1. [Open Windows Security]
2. [Perform a Full Scan and review Protection History]
3. [Simulate downloading the EICAR test file to see Defender in action]

---

## ✅ Verification Checklist

- [ ] Network adapter disabled successfully
- [ ] Startup items reviewed
- [ ] Defender scan completed
- [ ] Understand the purpose of the EICAR test file
